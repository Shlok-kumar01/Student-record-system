# Student Record System

**Author:**  Rahul Verma | Shlok Kumar

A CLI-based Student Record Management System built using the **C programming language**. 

This project simulates a simple **digital student record system** where users can manage student information through a terminal interface.

## Features

- **Add new student records**
- **View all stored records**
- **Search student by roll number**
- **Update existing student details**
- **Delete student records**
- **Calculate student results**

## Concepts used

This project demonstrates several important C programming concepts:

- **Structures (`struct`)** for representing student data
- **Functions** for modular and reusable code
- **Conditional statements (`if, switch`)**
- **Loops (`for, while`)**
- **Input validation**
- **File handling** for persistent storage

## Purpose of the Project

This project was created to practice:

- Core C programming concepts
- Data management using structures
- Building CLI applications
- Problem solving and program design
